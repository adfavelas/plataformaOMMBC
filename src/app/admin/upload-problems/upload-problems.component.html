Agregar Problemas

Para agregar nuevos problemas a la base de datos, por favor utiliza el botón que se encuentra debajo de este párrafo ("Archivo (CSV)") para seleccionar el archivo csv con los problemas que se deseen agregar.

El archivo .csv deberá contar con las siguientes columnas:

idOmmbc, area, topic, Tema, level, problemNumber, problemDescription, image, tip, solution, origin, keywords, notes, list, y advanced.

De no contener estos campos, la consistencia de la información podría verse comprometida.
